# puppeteer-rthk-traffic-news

> Using Puppeteer to Retrieve RTHK Traffic News Website in JSON Format.

Testing Purpose Only!

## Prerequisite
Node.js 8.x or Later

## To Execute

```sh
npm install
node index.js
```

## Format
JSON Array
Properties
1. news - Traffic News Content
2. date - Date in ISO String
